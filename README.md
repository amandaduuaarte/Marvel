## 🚀 Instalando Marvel

Para instalar o Marvel, siga estas etapas:

```
yarn ou npm
```

macOS:
```
yarn run android ou yarn run ios
```

Windows:
```
yarn run android
```

## 📫 Tudo que foi usado no desenvolvimento

1. TypeScript.
2. Styled-components.
3. React-Navigator.
4. Axios.
